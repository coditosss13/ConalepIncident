name: Stack Tecnológico Conalep
description: Tecnologías seleccionadas para el proyecto
type: reference
---

**Backend:**
- Node.js + Express
- MySQL (schema existente en BDexample.sql)
- JWT para autenticación
- Multer para archivos
- PDFKit para generación de PDFs

**Frontend:**
- React 18+
- React Router para navegación
- Context API + useReducer para state management
- Tailwind CSS para estilos
- Axios para peticiones HTTP
- React Query para cache de datos
- React Hook Form + Zod para formularios

**Paleta de colores:**
- Primario: #007d68 (Verde institucional)
- Secundario: #FFFFFF (Blanco)
- Accents: Tonos derivados del verde
